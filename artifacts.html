Structures: Extension Definitions

These define constraints on FHIR data types for systems conforming to this implementation guide

Case Based Surveillance Sex Assigned at Birth Extension

A code classifying the person’s sex assigned at birth.
Case Based Surveillance CDC Address Use Extension

Indicates the use of the address as per information surveillance program.
Case Based Surveillance Case Class Status

A code denoting the status of the case/event as suspect, probable, confirmed, or not a case per CSTE/CDC/surveillance case definitions.
Case Based Surveillance Diagnosis Date Extension

A dateTime indicating when the condition of interest was diagnosed.
Case Based Surveillance Ethnicity Extension

Concepts classifying the person into a named category of humans sharing common history, traits, geographical origin or nationality. The ethnicity codes used to represent these concepts are based upon the CDC ethnicity and Ethnicity Code Set Version 1.0 which includes over 900 concepts for representing race and ethnicity of which 43 reference ethnicity. The ethnicity concepts are grouped by and pre-mapped to the 2 OMB ethnicity categories: - Hispanic or Latino - Not Hispanic or Latino. This extension is based on the equivalent US Core extension though utilizes the public health focused PHIN Value Set and limiting data based on established standards in public health case based surveillance messaging.
Case Based Surveillance Race Extension

Concepts classifying the person into a named category of humans sharing common history, traits, geographical origin or nationality.
Case Based Surveillance Specimen Role Extension

Indicates the role of the sample.
Case Based Surveillance Illness Duration

Illness duration to obscure PHII in place of absokute dates.
Case Based Surveillance Program Specific Time Window

Time window of interest in relationship to a specified event as determined by individual case based surveillance programs.
Case Based Surveillance Reason Not Vaccinated Per ACIP Recommendations

Reason the subject was not vaccinated per ACIP recommendations.
Case Based Surveillance Related Case Extension

A list of strings with relevant case numbers and a type element that specifies the nature of the relation.

Terminology: Value Sets

These define sets of codes used by systems conforming to this implementation guide

Additional codes for Address.use.

Codes for Address.use which are used as slice discriminators to capture address history at relevant points in time to case surveillance.
CBS Case Notification Panel Value Set

Codes found in the case notification panel that are otherwise not captured in other CBS profiles.
Combined ValueSet for Lab DiagnosticReport Code.

Codes for Lab Test Type. The codes are from two PHINVADS valuesets to support labs test repeating group
Combined ValueSet for Lab Observation Result.

Codes for Lab Observations Result Value. The codes are from two PHINVADS valuesets to support lab results in MMG and Lab Repeating Group
Combined ValueSet for Lab Observation Code.

Codes for Lab related Observations. The codes are to support labs in MMG and Lab Repeating Group
CBS Related Case Type Value Set

Codes found in the case notification panel that are otherwise not captured in other CBS profiles.
STD Sexual History and Drug Use Questionnaire Panel

This is an example valueset for how to setup coding for an Epi Questionnaire Panel using a selection of codes from the STD MMG.
CBS Case Social Determinants of Health Category Value Set

Categories for social determinants of health and related elements of social history which may be captured in the SDOH profile.
CBS Case Social Determinants of Health Codes Value Set

Codes for social determinants of health and related elements of social history which may be captured in the SDOH profile.
CBS Time Window Relative To Value Set

Codes for specific resource elements referenced by the CBS Program Specific Time Window extension, to convey to which element in the referenced resource a time window is relative.

Terminology: Code Systems

These define new code systems used by systems conforming to this implementation guide

Case Based Surveillance temporary/placeholder code system.

Code system defined as placeholders for items that are not defined in existing systems.
