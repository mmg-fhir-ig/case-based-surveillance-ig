This page provides a list of the FHIR artifacts defined as part of this implementation guide.

CBS Profiles

Case Based Surveillance Profiles

Case Based Surveillance Cause of Death Profile

Defines constraints on on the Observation resource for supporting public health reporting workflows involving the cause of death of the subject.
Case Based Surveillance Composition Profile

Defines constraints to the Composition resource in order to meet the needs of public health surveillance programs.
Case Based Surveillance Document Profile

Defines a document type bundle that contains all related profiles.
Case Based Surveillance Questionnaire Profile

Defines constraints on the Questionnaire resource to set the minimum expected details to be provided as part of case based surveillance program’s questionnaire/survey data.
Case Based Surveillance Social Determinants of Health Profile

Defines constraints on the Observation resource to capture subject’s Social Determinants of Health in order to meet the needs of public health surveillance programs.
Case Based Surveillance Specimen Profile

Defines constraints on the Specimen resource in order to meet the needs of public health surveillance programs.
Case Based Surveillance Subject Vaccinated per ACIP Recommendations Profile

Defines constraints on the Observation resource to capture whether subject was vaccinated per ACIP Recommendations in order to meet the needs of public health surveillance programs.

Case Based Surveillance Age at Case Investigation

Case Based Surveillance Binational Reporting Criteria

Defines constraints on CBS Case Notification Panel to represent the Binational Reporting Criteria of the case notification panel in public health surveillance programs.
Case Based Surveillance Case Notification Panel Profile

Defines constraints on Observation to represent the case notification panel in public health surveillance programs.
Case Based Surveillance Case Outbreak

Defines constraints on CBS Case Notification Panel to represent the Case Outbreak section of the case notification panel in public health surveillance programs.
Case Based Surveillance Date of Initial Report

Case Based Surveillance Date of First Report to Public Health Department

Defines constraints on CBS Case Notification Panel to represent the date reported of the case notification panel in public health surveillance programs.
Case Based Surveillance Earliest Date Reported

Case Based Surveillance Earliest Date Reported to State

Case Based Surveillance Exposure Observation Profile

Case Based Surveillance Immediate National Notifiable Condition

Case Based Surveillance Case Investigation Start Date

Defines constraints on CBS Case Notification Panel to represent the case investigation start date of the case notification panel in public health surveillance programs.
Case Based Surveillance Jurisdiction Code

Defines constraints on CBS Case Notification Panel to represent the jurisdiction code of the case notification panel in public health surveillance programs.
Case Based Surveillance Morbidity and Mortality Weekly Report (MMWR) Elements

Defines constraints on CBS Case Notification Panel to represent the MMWR elements of the case notification panel in public health surveillance programs.
Case Based Surveillance National Reporting Jurisdiction

Case Based Surveillance Pregnancy Status

Case Based Surveillance Reporting County

Defines constraints on CBS Case Notification Panel to represent the reporting county of the case notification panel in public health surveillance programs.
Case Based Surveillance Reporting State

Defines constraints on CBS Case Notification Panel to represent the transmission mode of the case notification panel in public health surveillance programs.

US CBS Profiles

United States Case Based Surveillance Profiles

US Case Based Surveillance Condition of Interest Profile

Defines constraints and extensions to the Condition resource in order to meet the needs of public health surveillance programs, further constrainting the US Core based ECR US Public Health profiles.
US Case Based Surveillance Hospitalization Encounter Profile

Defines constraints on the ECR US Public Health Encounter to represent hospitalization due to a condition of interest for public health case based surveillance programs.
US Case Based Surveillance Immunization Profile

Defines further constraints on the US Core Immunization profile to capture subject vaccination records in order to meet the needs of public health surveillance programs.
US Case Based Surveillance Lab Diagnostic Report

Defines a DiagnosticReport resource profile in order to meet the needs of public health surveillance programs.
US Case Based Surveillance Lab Observation

Defines constraints on the Observation resource in order to meet the needs of lab reporting in public health surveillance programs.
US Case Based Surveillance Patient Profile

Defines further constraints and extensions to the ECR US Public Health profile in order to meet the needs of case based surveillance programs while maintaining adherence to US Core standards.
US Case Based Surveillance Performing Laboratory Profile

Defines further constraints and extensions to the ECR US Public Health Organization profile in order to meet the needs of case based surveillance programs while maintaining adherence to US Core standards.
US Case Based Surveillance Reporting Source Organization Profile

Defines constraints to the Organization resource in order to meet the needs of public health surveillance programs regarding the reporting source.
US Case Based Surveillance Travel History Profile

Defines constraints on the ECR US PH Travel History profile to capture additional data in order to meet the needs of public health case based surveillance programs.

CBS Extensions

Case Based Surveillance Extensions

Case Based Surveillance CDC Address Use Extension

Indicates the use of the address as per information surveillance program.
Case Based Surveillance Case Class Status

A code denoting the status of the case/event as suspect, probable, confirmed, or not a case per CSTE/CDC/surveillance case definitions.
Case Based Surveillance Specimen Role Extension

Indicates the role of the sample.
Case Based Surveillance Illness Duration

Illness duration to obscure PHII in place of absokute dates.
Case Based Surveillance Related Case Extension

A list of strings with relevant case numbers and a type element that specifies the nature of the relation.

US CBS Extensions

United States Case Based Surveillance Extensions

US Case Based Surveillance Program Specific Time Window

Time window of interest in relationship to a specified event as determined by individual case based surveillance programs.
US Case Based Surveillance Period and Mode of Travel Extension

An extension capturing the circumstances of travel to a destination.

Terminology: Value Sets

These define sets of codes used by systems conforming to this implementation guide

Additional codes for Address.use.

Codes for Address.use which are used as slice discriminators to capture address history at relevant points in time to case surveillance.
CBS Case Notification Panel Value Set

Codes found in the case notification panel that are otherwise not captured in other CBS profiles.
Combined ValueSet for Lab DiagnosticReport Code.

Codes for Lab Test Type. The codes are from two PHINVADS valuesets to support labs test repeating group
CBS Related Case Type Value Set

Codes found in the case notification panel that are otherwise not captured in other CBS profiles.
STD Sexual History and Drug Use Questionnaire Panel

This is an example valueset for how to setup coding for an Epi Questionnaire Panel using a selection of codes from the STD MMG.
CBS Case Social Determinants of Health Category Value Set

Categories for social determinants of health and related elements of social history which may be captured in the SDOH profile.
CBS Case Social Determinants of Health Codes Value Set

Codes for social determinants of health and related elements of social history which may be captured in the SDOH profile.
CBS Time Window Relative To Value Set

Codes for specific resource elements referenced by the CBS Program Specific Time Window extension, to convey to which element in the referenced resource a time window is relative.

Terminology: Code Systems

These define new code systems used by systems conforming to this implementation guide

Case Based Surveillance temporary/placeholder code system.

Code system defined as placeholders for items that are not defined in existing systems.
