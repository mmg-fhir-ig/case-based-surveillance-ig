Structures: Resource Profiles

These define constraints on FHIR resources for systems conforming to this implementation guide

Case Based Surveillance Epi Questionnaire Panel

Defines constraints on Observation to represent structured resources for the collection and organization of epidemiological questions.
Case Based Surveillance Case Notification Panel Member

Defines constraints on the CBS Case Notification Panel Profile for defining the member resources of the case notification panel in public health surveillance programs.
Case Based Surveillance Cause of Death Profile

Defines constraints on on the Observation resource for supporting public health reporting workflows involving the cause of death of the subject.
Case Based Surveillance Co-Condition Profile

Defines constraints on on the Condition resource for any additional conditions beyond the condition of interest for surveillance. The requirements for these conditions are less strict structurally, and warrant a simpler profile.
Case Based Surveillance Composition Profile

Defines constraints to the Composition resource in order to meet the needs of public health surveillance programs.
Case Based Surveillance Condition of Interest Profile

Defines constraints and extensions to the Condition resource in order to meet the needs of public health surveillance programs while providing as much alignment with US Core requirements as possible.
Case Based Surveillance Exposure Observation Profile

Defines constraints to the CBS Case Notification Panel Member profile to represent the exposure observations of the case notification panel.
Case Based Surveillance Hospitalization Encounter Profile

Defines constraints on Encounter to represent hospitalization due to a condition of interest in public health surveillance programs.
Case Based Surveillance Morbidity and Mortality Weekly Report (MMWR) Elements

Defines constraints on CBS Case Notification Panel to represent the MMWR elements of the case notification panel in public health surveillance programs.

Case Based Surveillance Diagnosis Date Extension

A dateTime indicating when the condition of interest was diagnosed.
Case Based Surveillance Ethnicity Extension

Concepts classifying the person into a named category of humans sharing common history, traits, geographical origin or nationality. The ethnicity codes used to represent these concepts are based upon the CDC ethnicity and Ethnicity Code Set Version 1.0 which includes over 900 concepts for representing race and ethnicity of which 43 reference ethnicity. The ethnicity concepts are grouped by and pre-mapped to the 2 OMB ethnicity categories: - Hispanic or Latino - Not Hispanic or Latino. This extension is based on the equivalent US Core extension though utilizes the public health focused PHIN Value Set and limiting data based on established standards in public health case based surveillance messaging.
